1분 기본 질문 프레임
  • 결론: 나는 어떤 사람인가
  • 근거: 어떤 경험에서 드러났나
  • 행동: 내가 어떻게 했나
  • 결과: 무엇을 만들었나
  • 연결: PE에서 어떻게 쓰이나
3분 직무 질문 프레임
  • 정의: 개념을 1-2문장으로 설명
  • 중요성: 품질, 수율, 제품 성능과 연결
  • PE 관점: 테스트 조건, 불량 분석, 데이터 해석
  • 내 경험: TCAD, SRAM/MBIST, PPA, Compression, 협업 중 하나
  • 마무리: 입사 후 기여 방식
좋은 PE 답변 문장
  • 제품 특성을 테스트 조건과 데이터로 해석하겠습니다.
  • 불량 원인을 단정하지 않고 조건을 나누어 재현하겠습니다.
  • 품질과 테스트 효율 사이의 균형을 데이터로 판단하겠습니다.
  • 설계·소자·공정·AE·P&T와 같은 문제 정의를 만들겠습니다.
